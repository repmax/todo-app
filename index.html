About

This is a barebones Todo web app. Add, complete, and remove tasks – it's that simple.

Your data stays private! Tasks are stored locally in your browser using IndexedDB.

This project was an experiment in building reactive PWAs with vanilla JavaScript (ES6 modules), Lit, and IndexedDB – no build tools or network requests needed. This approach is not only great for creating transparent and secure web apps but also likewise browser extensions.

Check out the code on GitHub ... or get on top of your todo list right now!
